Description

This vulnerability allows attackers to interfere with database queries.

Example Payloads:

  • ' OR '1'='1 - Bypass authentication
  • ' UNION SELECT * FROM users-- - Extract all data
  • ' DROP TABLE users-- - Delete data

Mitigation: Use parameterized queries (prepared statements).
